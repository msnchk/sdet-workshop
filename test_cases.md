# Тест-кейсы для проверки функционала банковского приложения

**Общие предусловия**<br>
Перед выполнением каждого теста необходимо:

- Настроить веб-драйвер для браузера Сhrome

- Удалить cookies

- Перейти по адресу: https://www.globalsqa.com/angularJs-protractor/BankingProject/#/manager

**Общие постусловия**<br>
После выполнения каждого теста необходимо:

- Завершить процесс веб-драйвера

- Удалить данные из памяти

## Тест-кейс 1: Добавление нового клиента в систему (Add Customer) - позитивный сценарий

**Шаги:**

1. На странице "Bank Manager" нажать кнопку "Add Customer"

2. Ввести сгенерированные тестовые данные:

- Post Code – случайный 10-значный номер

- First Name – имя, сгенерированное на основе Post Code

- Last Name – случайная фамилия

4. Нажать кнопку "Add Customer"

5. Если появился алерт – закрыть его

6. Перейти на страницу списка клиентов

**Ожидаемый результат:**

- Появляется alert: "Customer added successfully"

- В списке клиентов отображается добавленный пользователь


## Тест-кейс 2: Попытка добавить клиента с незаполненными полями (Add Customer) - негативный сценарий

**Шаги:**

1. На странице "Bank Manager" нажать кнопку "Add Customer"

2. Поочередно оставить одно из полей пустым ("Post Code", "First Name", "Last Name").
Дальнейшие шаги повторяются для трех наборов некорректных данных (два поля заполнены, одно пустое).

3. Нажать кнопку "Add Customer" 

4. Закрыть появившийся алерт (если есть)

5. Перейти на страницу списка клиентов

**Ожидаемый результат:**

- Нет алерта "Customer added successfully"

- В списке нет добавленного пользователя

## Тест-кейс 3: Сортировка клиентов по имени (First Name)

**Шаги:**

1. Перейти в раздел "Customers"

2. Нажать на заголовок столбца "First Name" (сортировка в обратном порядке)

3. Собрать список имен и сравнить с ожидаемым порядком

4. Нажать заголовок снова (сортировка по алфавиту)

5. Снова сравнить порядок имен

**Ожидаемый результат:**

- Имена сортируются сначала в обратном порядке, затем по алфавиту

## Тест-кейс 4: Удаление клиента

**Шаги:**

1. Перейти в раздел "Customers"

2. Получить список всех имен

3. Определить имя для удаления (по средней длине)

4. Нажать кнопку "Delete" для выбранного клиента

5. Обновить список

**Ожидаемый результат:**

- Клиент успешно удален и не отображается в таблице
